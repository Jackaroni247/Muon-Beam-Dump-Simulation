# Muon-Beam-Dump-Simulation
The source code for our physic club's BL4S simulation. This is run with Geant4 and simulates the collision of a muon beam wih a customizable beam dump

Includes a sample image of a graph created with the data from the program

To run yourself simply compile using cmake and run the executable with or without the vis macro. The program is set up to allow multithreading for high number of events

Thank you to the Geant4 collaboration for making this wonderful tool avilable
